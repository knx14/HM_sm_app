Step B-6：測定画面 — エリア判定ロジックのスキップ

MeasurementSessionScreen のStep3（本測定）における「圃場内外判定（maps_toolkit）→ 圃場外なら測定開始禁止」 のロジックを、選択中の圃場が仮登録圃場（boundaryPolygon.isEmpty）の場合はスキップする。 

// MeasurementSessionScreen 内、圃場内外判定箇所
bool _isInsideOrEdge(LatLng point) {
  final farm = _selectedFarm;
  if (farm == null) return false;

  // 仮登録圃場（境界情報なし）はエリア判定自体を行わない
  if (farm.boundaryPolygon.isEmpty) {
    return true; // 常に測定可能とみなす
  }

  final containment = PolygonUtil.containsLocation(point, farm.boundaryPolygon, false);
  final onEdge = PolygonUtil.isLocationOnEdge(point, farm.boundaryPolygon, 1e-6);
  return containment || onEdge;
}
  • 本測定画面で、赤い圃場境界線が「仮登録圃場では表示されない」ことをUI上でも明示（境界未設定のバナー等）
  • 仮登録圃場選択時は「測定開始」ボタンが常に有効であることを確認
  • 境界情報がある通常の圃場では、従来通り圃場外判定が機能することを回帰確認
仮登録圃場はGPS判定を行わないため、誤って別の畑で測定してしまうリスクがある。 測定開始前の確認ダイアログに「この圃場は仮登録のため、位置情報による誤測定防止が働きません」の注意文言を追加することを推奨する（必須ではないが強く推奨）。

// app/Support/SoilParameterUnits.php（確定版・17種）
class SoilParameterUnits
{
    private const MAP = [
        'B'                => 'mg/kg',
        'CaO'              => 'mg/kg',
        'CEC'              => 'cmol/kg',
        'Cu'               => 'mg/kg',
        'Fe'               => 'mg/kg',
        'FU'               => null,
        'K2O'              => 'mg/kg',
        'MgO'              => 'mg/kg',
        'Mn'               => 'mg/kg',
        'NH4-N'            => 'mg/kg',
        'NO3-N'            => 'mg/kg',
        'P2O5'             => 'mg/kg',
        'PA'               => null,
        'SiO2'             => 'mg/kg',
        'Zn'               => 'mg/kg',
        '易還元性マンガン'  => 'mg/kg',
        '遊離酸化鉄'        => 'mg/kg',
    ];

    public static function unitFor(string $parameterName): ?string
    {
        if (!array_key_exists($parameterName, self::MAP)) {
            throw new \InvalidArgumentException("Unknown parameter_name: {$parameterName}");
        }
        return self::MAP[$parameterName];
    }

    public static function allParameterNames(): array
    {
        return array_keys(self::MAP);
    }
}
17種すべてのparameter_name/unitが確定したため、C-3-3（管理画面フォーム拡張）は17種すべてを対象に実装してよい。 以前の版にあった「まずCEC/CaO/K2O/MgOの4種に限定する」という制限は解除する。 ただしFU・PAは単位表記がないため、フォーム上の単位表示欄は空欄のまま送信できるようにし、 unitFor()がnullを返すケースとしてresult_values.unitにnull（または空文字）が正しく保存されることを確認する。
