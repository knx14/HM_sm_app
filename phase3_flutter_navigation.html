01ルーター定義の作成 router

現行の IndexedStack + BottomNav によるタブ切り替えを廃止し、 名前付きルート（Named Routes） によるスタックナビゲーションに移行します。 すべてのルート名をここで一元管理することで、 画面追加時に変更箇所が明確になります。

1

lib/router/app_router.dart を新規作成

lib/router/app_router.dart（新規作成）

// 各画面をインポート（実際のパスはプロジェクトに合わせて修正）
import '../screens/home/home_screen.dart';
import '../screens/sync/sync_screen.dart';
import '../screens/settings/settings_screen.dart';
import '../screens/help/help_screen.dart';
// 既存画面
import '../screens/measurement/measurement_session_screen.dart';
import '../screens/farm/farm_screen.dart';

/// アプリ内のすべての名前付きルートを一元管理する。
/// 画面間遷移は必ずここで定義したルート名を使うこと。
class AppRouter {
  // ── ルート名定数 ──────────────────────────────────────────
  static const String home        = '/';
  static const String measurement  = '/measurement';
  static const String farmList     = '/farms';
  static const String sync         = '/sync';
  static const String settings     = '/settings';
  static const String help         = '/help';

  // ── ルートテーブル ────────────────────────────────────────
  static Map<String, WidgetBuilder> get routes => {
    home:       (context) => const HomeScreen(),
    measurement:(context) => const MeasurementSessionScreen(),
    farmList:   (context) => const FarmScreen(),
    sync:       (context) => const SyncScreen(),
    settings:   (context) => const SettingsScreen(),
    help:       (context) => const HelpScreen(),
  };
}
既存画面のインポートパスについて。 measurement_session_screen.dart と farm_screen.dart のパスは 実際のプロジェクト構造に合わせてください。 find lib -name "*.dart" の結果を参照して正確なパスを使います。

class _HomeButton extends StatelessWidget {
  const _HomeButton({
    required this.label,
    required this.icon,
    required this.color,
    required this.onTap,
    this.subtitle,
  });

  final String   label;
  final String?  subtitle;
  final IconData icon;
  final Color    color;
  final VoidCallback onTap;

  @override
  Widget build(BuildContext context) {
    return Material(
      color: color,
      borderRadius: BorderRadius.circular(12),
      child: InkWell(
        onTap: onTap,
        borderRadius: BorderRadius.circular(12),
        child: Container(
          alignment: Alignment.center,
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Icon(icon, color: Colors.white, size: 32),
              const SizedBox(height: 8),
              Text(label,
                  style: const TextStyle(
                      color: Colors.white,
                      fontSize: 18,
                      fontWeight: FontWeight.bold)),
              if (subtitle != null)
                Text(subtitle!,
                    style: TextStyle(
                        color: Colors.white.withOpacity(0.8),
                        fontSize: 12)),
            ],
          ),
        ),
      ),
    );
  }
}

lib/screens/sync/sync_screen.dart を新規作成

lib/screens/sync/sync_screen.dartimport 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import '../../providers/pending_uploads_provider.dart';

class SyncScreen extends StatefulWidget {
  const SyncScreen({super.key});

  @override
  State<SyncScreen> createState() => _SyncScreenState();
}

class _SyncScreenState extends State<SyncScreen> {
  final Set<int> _selected = {};  // 選択中のインデックス

  @override
  Widget build(BuildContext context) {
    final provider = context.watch<PendingUploadsProvider>();
    final items    = provider.pendingItems;   // List<PendingUploadItem>

    return Scaffold(
      appBar: AppBar(
        title: const Text('同期'),
        backgroundColor: const Color(0xFFB07820),
        foregroundColor: Colors.white,
      ),
      body: items.isEmpty
          ? const Center(child: Text('未アップロードデータはありません'))
          : Column(
              children: [
                // 件数 + 最終同期日時
                _buildHeader(context, provider, items),
                // リスト
                Expanded(
                  child: ListView.separated(
                    itemCount: items.length,
                    separatorBuilder: (_, __) =>
                        const Divider(height: 1),
                    itemBuilder: (context, i) =>
                        _buildItem(context, provider, items[i], i),
                  ),
                ),
                // 下部ボタン
                _buildBottomBar(context, provider),
              ],
            ),
    );
  }

  Widget _buildHeader(BuildContext ctx, PendingUploadsProvider prov,
      List items) {
    return Container(
      padding: const EdgeInsets.all(16),
      color: const Color(0xFFFEF0D8),
      child: Row(children: [
        Text('未同期 ${items.length} 件',
            style: const TextStyle(fontWeight: FontWeight.bold)),
        const Spacer(),
        TextButton(
          onPressed: () => setState(() {
            _selected.length == items.length
                ? _selected.clear()
                : _selected.addAll(
                    List.generate(items.length, (i) => i));
          }),
          child: Text(_selected.length == items.length
              ? '全解除' : '全選択（${items.length} 件）'),
        ),
      ]),
    );
  }

  Widget _buildItem(BuildContext ctx, PendingUploadsProvider prov,
      dynamic item, int i) {
    // PendingUploadItem のフィールド名は既存実装に合わせて調整
    return CheckboxListTile(
      value: _selected.contains(i),
      onChanged: (v) => setState(() =>
          v! ? _selected.add(i) : _selected.remove(i)),
      title: Text(item.farmName ?? '圃場不明'),
      subtitle: Text(item.timestamp ?? ''),
      secondary: Icon(
        item.hasError ? Icons.warning_amber_rounded : Icons.cloud_upload_outlined,
        color: item.hasError ? Colors.red : Colors.grey,
      ),
    );
  }

  Widget _buildBottomBar(BuildContext ctx, PendingUploadsProvider prov) {
    return Container(
      padding: const EdgeInsets.all(16),
      child: Row(children: [
        OutlinedButton(
          onPressed: _selected.isEmpty ? null :
              () => prov.deleteSelected(_selected.toList()),
          child: const Text('選択を削除'),
        ),
        const SizedBox(width: 12),
        Expanded(
          child: ElevatedButton(
            onPressed: _selected.isEmpty ? null :
                () => prov.syncSelected(_selected.toList()),
            style: ElevatedButton.styleFrom(
                backgroundColor: const Color(0xFFB07820)),
            child: const Text('同期',
                style: TextStyle(color: Colors.white)),
          ),
        ),
      ]),
    );
  }
}
PendingUploadItem のフィールド名について。 item.farmName / item.timestamp / item.hasError は 既存の PendingUploadItem クラスのフィールド名に合わせてください。 また provider.deleteSelected() / provider.syncSelected() は 既存の未アップロード一覧画面で使っているメソッドをそのまま流用します。 既存の「未アップロード一覧」画面のコードを参照して実際のメソッド名に置き換えてください。

              // 既存のサインアウト処理を呼ぶ（UserProvider.signOut() など）

              if (context.mounted) {

06MainScaffold / BottomNav の廃止 remove bottom nav

7

MainScaffold から BottomNavigationBar を削除

lib/widgets/main_scaffold/main_scaffold.dart（既存ファイルを編集）

現行の MainScaffold は IndexedStack + BottomNavigationBar で3タブを管理しています。このファイルを開いて以下の変更を行います。

  • IndexedStack および BottomNavigationBar に関するコードをすべて削除
  • _currentIndex などのタブ管理 state を削除
  • ファイル全体が不要になる場合はファイルごと削除してもよい（後述の main.dart 変更で不要になる）
「MainScaffold」自体を削除するアプローチを推奨します。 MainScaffold は BottomNav のためだけに存在するラッパーです。 BottomNav を廃止した後は main.dart から直接 HomeScreen を initialRoute で表示すれば済みます。 削除後に main.dart から MainScaffold の参照を除去してください。
8

main.dart のエントリーポイントを変更

lib/main.dart（既存ファイルを編集）

MaterialApp の設定を以下のように変更します。 home: MainScaffold() から initialRoute + routes 方式へ移行します。 

lib/main.dart — MaterialApp の変更箇所（変更前 → 変更後）// 変更前（BottomNav 時代）
MaterialApp(
  // ...
  home: MainScaffold(),  // ← 削除
)

// 変更後（AppRouter 使用）
import 'router/app_router.dart';  // ← 追加

MaterialApp(
  // theme, locale など既存設定はそのまま維持
  initialRoute: AppRouter.home,
  routes: AppRouter.routes,
)
SplashScreen / AuthGate の扱い。 現行では起動時に SplashScreen で認証状態を確認しています。 initialRoute が HomeScreen に直接飛ぶと 未認証状態でホームが表示される可能性があります。 既存の認証チェックロジック（SplashScreen → HomeScreen）を 以下のいずれかで維持してください：
  • 推奨: initialRoute: '/splash' として SplashScreen を先頭に残す
  • または HomeScreen の initState で認証確認 → 未認証なら Login へリダイレクト
実際の認証フローは SplashScreen の既存コードを参照して判断してください。
