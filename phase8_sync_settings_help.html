現行の SyncScreen は StatefulWidget です。 State クラスに以下のフィールドを追加してください。 

SyncScreen の State に追加するフィールドとメソッド// 追加するフィールド
bool   _isSyncing     = false;
int    _syncedCount   = 0;
int    _totalToSync   = 0;
String _lastSyncedAt  = '';

@override
void initState() {
  super.initState();
  _loadLastSyncedAt();
}

// 最終同期日時を SharedPreferences から読み込む
Future<void> _loadLastSyncedAt() async {
  final prefs = await SharedPreferences.getInstance();
  setState(() {
    _lastSyncedAt = prefs.getString('last_synced_at') ?? '未同期';
  });
}

// 最終同期日時を保存する（同期完了時に呼ぶ）
Future<void> _saveLastSyncedAt() async {
  final prefs = await SharedPreferences.getInstance();
  final now   = DateTime.now();
  final label = '${now.month}/${now.day} ${now.hour.toString().padLeft(2,"0")}:${now.minute.toString().padLeft(2,"0")}';
  await prefs.setString('last_synced_at', label);
  setState(() => _lastSyncedAt = label);
}

// 選択した測定データを同期する（既存の同期処理を拡張）
Future<void> _syncSelected(PendingUploadStore store) async {
  if (_selected.isEmpty) return;
  setState(() {
    _isSyncing   = true;
    _syncedCount = 0;
    _totalToSync = _selected.length;
  });

  for (final idx in _selected.toList()) {
    try {
      // 既存の同期処理メソッドを1件ずつ呼ぶ
      // store.syncItem(idx) または相当メソッドに合わせること
      await store.syncSelected([idx]);
      setState(() => _syncedCount++);
    } catch (_) {
      // エラーが出ても次のアイテムに進む
    }
  }

  setState(() => _isSyncing = false);
  _selected.clear();
  await _saveLastSyncedAt();
}
PendingUploadStore の syncSelected メソッド名について。 Phase 3 の実装では syncSelected(List<int>) の形で1件ずつ呼ぶ想定でしたが、 実際の PendingUploadStore のメソッドシグネチャを確認して合わせてください。 grep -n "Future.*sync\|void.*sync" $(find lib -name "*pending_upload*store*") で確認できます。

HomeScreen バッジ更新の動作確認

確認作業（コード変更なし）

HomeScreen の同期ボタンバッジは PendingUploadStore の件数を監視しています。 _syncSelected 実行後に PendingUploadStore の件数が減れば 自動的にバッジが更新されるはずです。

# HomeScreen が PendingUploadStore をどう監視しているか確認 $ grep -n "PendingUploadStore\|pendingCount\|watch\|listen" \ lib/screens/home/home_screen.dart | head -10
バッジが更新されない場合。 PendingUploadStore が ChangeNotifier であり、 同期後に notifyListeners() が呼ばれていることを確認してください。 呼ばれていない場合は syncSelected の完了後に notifyListeners() を追加します。
